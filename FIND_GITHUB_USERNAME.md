## Quick Ways to Find Your GitHub Username:

### Method 1: Check Your GitHub Profile URL (Easiest!)

1. **Go to GitHub.com** and **sign in**
2. **Click your profile picture** (top right corner)
3. **Click "Your profile"**
4. **Look at the URL** in your browser's address bar
   - It will look like: `https://github.com/YOUR_USERNAME`
   - **YOUR_USERNAME** is what you need!

**Example:**
- If URL is: `https://github.com/recorpproduction`
- Your username is: `recorpproduction`

---

### Method 2: Check Your Email's GitHub Account

1. **Go to GitHub.com**
2. **Sign in** with `recorpproduction@gmail.com`
3. **Click your profile picture** (top right)
4. **Your username is shown** under your name

---

### Method 3: Check Repository URL

1. **Go to any of your repositories** on GitHub
2. **Look at the URL:**
   - `https://github.com/YOUR_USERNAME/repository-name`
   - The part after `github.com/` and before `/repository-name` is your username

---

### Method 4: Check Settings

1. **Go to GitHub.com** ‚Üí **Sign in**
2. **Click your profile picture** (top right)
3. **Click "Settings"**
4. **Your username is shown** at the top of the settings page

---

## Once You Have Your Username:

1. **Open `index.html`** in your code editor
2. **Find this line** (around line 642):

```javascript
owner: "YOUR_GITHUB_USERNAME", // TODO: Replace with your GitHub username
```

3. **Replace `YOUR_GITHUB_USERNAME`** with your actual username
   - Example: `owner: "recorpproduction",`

4. **Save the file**

---

## Common Examples:

- If your GitHub URL is `github.com/johnsmith` ‚Üí username is `johnsmith`
- If your GitHub URL is `github.com/recorpproduction` ‚Üí username is `recorpproduction`
- If your GitHub URL is `github.com/company-name` ‚Üí username is `company-name`

**The username is the part right after `github.com/`!**
